Test.csv CSV de una sola columna con campo de identificación. Id es una cadena de 16 caracteres.
Recognition_solution_v2.1.csv CSV con tres columnas: id(cadena de 16 caracteres), landmarks(lista de identificadores enteros de puntos de referencia separados por espacios, o vacía si no se muestra ningún punto de referencia del conjunto de datos), Usage(ya sea "Público" o "Privado", indicando a qué subconjunto pertenece la imagen).
Retrieval_solution_v2.1.csv CSV con tres columnas: id(cadena de 16 caracteres), images(lista de ID de imágenes de índice de cadena separadas por espacios, o "Ninguno" si se ignora esta imagen), Usage(ya sea "Público" o "Privado", indicando a qué subconjunto pertenece la imagen).
