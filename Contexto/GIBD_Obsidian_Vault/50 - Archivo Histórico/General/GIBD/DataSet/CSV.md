# CSV

Ruta interna: `GIBD/DataSet/CSV.docx`

---

El archivo adjuntado en la carpeta forma parte del conjunto de datos del desafío Landmark Retrieval 2019 de Kaggle. 
Contiene 117577 filas donde se encuentra una lista de IDs de imágenes.

Conjunto de datos
Las imágenes de consulta se listan en [nombre del archivo] test.csv, mientras que las imágenes de "índice" de las que se recuperan se listan en [nombre del archivo index.csv]. Cada imagen tiene un [nombre del archivo] único id.
Datos de prueba
Consta de dos etapas. Hay un conjunto de pruebas para la primera etapa y otro conjunto de pruebas para la segunda.
La test.csv primera etapa. Las imágenes están disponibles mediante URL y deben descargarse.
Script de Python: https://www.kaggle.com/code/tobwey/landmark-retrieval-challenge-image-downloader/script 
Los datos de prueba para la segunda etapa se pueden obtener a través de CVDF 

Hay 117.577 imágenes en el test conjunto.
Datos del índice
Consta de dos etapas. Hay un índice establecido para la primera etapa y otro diferente para la segunda.
La index.csv primera etapa. Las imágenes están disponibles mediante URL y deben descargarse.
Script de Python: https://www.kaggle.com/code/tobwey/landmark-retrieval-challenge-image-downloader 
Los datos del índice para la segunda etapa se pueden obtener a través de CVDF

Hay 761.757 imágenes en el index conjunto.
