Curso complementario Machine Learning 
Introducción:
Se van a ir ampliando conceptos como el uso de python (principal lenguaje de programación de machine Learning), así como de todas las librerías que se usan en esta área como son numpy, pandas, sklearn,etc. 
Cada algoritmo estará organizado en 3 videos, uno que es toda la parte teórica del mismo, en el segundo una explicación con scikit learn, el último video una práctica del algoritmo en cuestión. 

Lenguajes de programación de machine learning:
¿Cuál es el mejor lenguaje? Cuando pensamos en ML, se nos viene a la mente Python, R, Matlab, Julia
Python: Es el gran líder en el ámbito de desarrollo de machine learning debido a la facilidad que tiene para escribir código, además tiene librerías específicas que hace que la computadora pueda aprender álgebra lineal y demás opciones matemáticas.
R: Está diseñado específicamente para la visualización estadística, se pueden construir algoritmos y fácil visualización con R studio, es una alternativa gratuita a los costosos software estadísticos como matlab. 
Matlab: Algoritmo, rápido, estable y muy seguro, asegurando la matemática compleja, encuentra camino en muchas aplicaciones
Julia: Está ganando un gran impulso entre desarrolladores, proporciona poderosísima interfaz gráfica que facilita el aprendizaje y la experiencia trabajando en esta área de la ciencia. 



IDE SPYDER:
Este ide es de código abierto, está escrito en python, orientado a investigación, creación de paquetes científicos. Cuenta con una interfaz de usuario multilingüe con finalización automática de código, está dirigido a científicos, ingenieros, analistas de datos pero no significa que los codificadores no podamos usar las funciones que trae. 
Tiene un explorador de variables, dónde podemos fácilmente visualizar las variables que hemos ido creando a lo largo del programa, tiene explorador de archivos que podemos ir viendo los archivos que hemos ido instalando además de un visor de ayuda. 
Durante el desarrollo podemos ir agregando puntos de interrupción para probar el código por partes. 
Comentario acerca de jupyer: Ni spyder ni jupyter es superior uno al otro, simplemente debemos elegir con cual nos sentimos más cómodos programando. 

Librería Numpy de python:
Proporciona una estructura de datos de matriz para poder ampliar la funcionalidad de los códigos que utilizamos.
Quiere decir Numerical Python, proporciona estructuras de datos de n dimensiones.
